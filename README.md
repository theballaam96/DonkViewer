# DonkViewer

DonkViewer is a web-based interactive viewer for exploring all 216 maps in Donkey Kong 64 with multiple display modes, including geometry and collision data, right in your browser.

## Live Demo

A hosted version of DonkViewer is available at:
https://theballaam96.github.io/DonkViewer/

You can open this in any modern web browser without any required installation. You must supply a DK64 ROM though to operate the viewer as the viewer pulls data from your ROM to generate it's output.

## Features

✔ Shows all 216 maps from DK64 with selectable view modes
✔ Geometry visualization
✔ Collision visualization
✔ Internal Element visualization
